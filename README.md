# CareerHub : Backend
CareerHub is a Job/Internship Platform with AI Magic for Fast Placements.
The Documentation of the API can be accessed at /docs endpoint

## Installation
1. Install Pipenv


```bash
pip3 install pipenv
```

2. Install Deps

```bash
pipenv install
```

3. Run

```bash
pipenv run uvicorn app:app --reload --host 0.0.0.0
```
